Ceramic Capacitor Code Decoder

100 nF
= 0.1 μF = 100,000 pF
Format: 3-digit code where first 2 digits = significant figures, 3rd digit = multiplier (number of zeros to add in pF).
Special: Code ending in 8 = ×0.01, ending in 9 = ×0.1. "R" replaces decimal point.
Examples: 104 = 100,000 pF = 100 nF | 472 = 4700 pF = 4.7 nF | 221 = 220 pF
