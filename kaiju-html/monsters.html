Implementation

Archetypes populate kaijuTypes[KAIJU_TYPE_COUNT] at startup via LoadKaijuArchetypes. Each slot can load its own .glb model (GOZURA uses assets/kaiju.glb; the in-repo asset is ps4_csis_godzilla_2.glb). If a model file is missing the slot stays selectable with modelLoaded=false and the game renders a procedural stand-in — drop the file in and re-launch, no code change needed.

Per-archetype tuning knobs: baseWidth (size after NormalizeModel) and yawOffsetDeg (one-edit fix for models that load facing the wrong way). The select-screen carousel pages with < / > or numeric keys 1 / 2 / 3.
