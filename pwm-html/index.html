RC Servo Control

Standard RC servos use PWM at 50 Hz (20ms period). Pulse width controls angle: 1000µs = 0°, 1500µs = 90° (center), 2000µs = 180°. Pulse range varies by servo — some use 500–2500µs.

Arduino Servo Library

#include <Servo.h>
Servo myServo;

void setup() {
  myServo.attach(9);  // PWM pin
}

void loop() {
  myServo.write(0);    // 0 degrees
  delay(1000);
  myServo.write(90);   // 90 degrees
  delay(1000);
  myServo.write(180);  // 180 degrees
  delay(1000);
}
