Gallery — The BBS Era in Images
Ward Christensen & the First BBS
Ward Christensen with the expansion card of CBBS, the world's first BBS, launched February 16, 1978 in Chicago during a blizzard.
Hayes 300 Baud Smartmodem
The Hayes Smartmodem — its AT command set became the universal standard for modem control, used by every BBS caller.
ANSI Art — The BBS Artscene
ANSI art created by BBS artists using IBM PC character set and color codes — the visual language of bulletin board culture.
FreeNet BBS Welcome Screen, 1994
A 1994 FreeNet bulletin board system welcome screen — the first thing callers saw when connecting via modem.
Neon BBS Welcome Screen
A typical 1990s BBS opening screen with ANSI graphics — sysops spent hours crafting these welcoming displays for callers.
BBS Login Screen
The Koala Country BBS ANSI login screen — users typed their handle and password to access message bases, files, and door games.
FidoNet Node List, June 1984
The hand-compiled list of Fido BBS systems from June 1984 — the earliest days of the global network that would grow to 35,000 nodes.
Amiga 3000 Running a BBS
An Amiga 3000UX running a two-line BBS — the Amiga's multitasking OS made it a popular platform for sysops managing multiple callers.
