What happens when you click Run?

  1. 1 Your browser sends the hostname + port to this server over HTTPS.
  2. 2 The server runs the tls-ca-fetch binary (Linux amd64) against the target host, capturing full stdout and stderr.
  3. 3 The full output and PEM file (if a CA cert was found) are returned to your browser in the response.
  4. 4 Nothing is stored server-side. No logs, no certificates, no request history.
