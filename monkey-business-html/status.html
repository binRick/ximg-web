What this means

The server pulls quotes for all 100 NASDAQ-100 tickers once per round. Yahoo is the primary feed (one batch call per 20 symbols, so 5 calls per round). Stooq is only touched when Yahoo throws, returns sparse data, or otherwise fails — in steady state its counters stay flat.

A provider is marked blocked when its most recent error matches a known rate-limit signature (Stooq's Exceeded the daily hits limit body, or any Yahoo 4xx). It clears as soon as a fresh call from that provider succeeds.

Counters reset on every container restart. The raw JSON is at /api/stats.
