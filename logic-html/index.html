Logic Gate Reference

AND

Y = A · B. HIGH only when all inputs HIGH. Used for masking, enabling signals. NAND is universal — any logic function can be built from NAND gates alone.

A=1 B=1 → Y=1   (otherwise 0)

OR

Y = A + B. HIGH when any input is HIGH. NOR is also universal. Used in flag detection, alarm systems, OR-wired buses.

A=0 B=0 → Y=0   (otherwise 1)

NOT (Inverter)

Y = Ā. Single input, inverted output. Fundamental building block. CMOS inverter: a PMOS and NMOS transistor in series.

A=0 → Y=1   A=1 → Y=0

NAND

Y = NOT(A · B). Universal gate. Cheaper to fabricate than AND (one fewer transistor layer). Used in SRAM cells, priority encoders.

A=1 B=1 → Y=0   (otherwise 1)

NOR

Y = NOT(A + B). Also universal. Used in SR latches (cross-coupled NOR). Output HIGH only when all inputs are LOW.

A=0 B=0 → Y=1   (otherwise 0)

XOR

Y = A ⊕ B. HIGH when inputs differ. Core of adders, checksum circuits, CRC generators. XOR with itself = 0, XOR with 1 = complement.

A≠B → Y=1   A=B → Y=0

XNOR

Y = NOT(A ⊕ B). HIGH when inputs are equal. Equality detector. Used in comparators, error-detection circuits.

A=B → Y=1   A≠B → Y=0

De Morgan's Laws

Fundamental identities for simplifying logic expressions:

NOT(A AND B) = (NOT A) OR (NOT B)

NOT(A OR B) = (NOT A) AND (NOT B)

Boolean Identities

Key rules for simplification:

A + 0 = A
 
A · 1 = A

A + 1 = 1
 
A · 0 = 0

A + A = A
 
A · A = A

A + Ā = 1
 
A · Ā = 0

Voltage Levels (TTL / CMOS)

TTL (5V): LOW <0.8V, HIGH >2.0V. Noise margin ~0.4V.
CMOS (3.3V): LOW <1.0V, HIGH >2.3V. Near-rail-to-rail swing.
LVTTL / LVCMOS: common in FPGAs and microcontrollers. Undefined region between thresholds should be avoided to prevent oscillation.
