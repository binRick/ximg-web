3-Band Resistors

3-band resistors have 20% tolerance and are the simplest type. The first two bands give the significant digits; the third band is the multiplier (power of 10). Formula: R = (B1×10 + B2) × multiplier

SMD Code Format Guide

3-digit (e.g. 103): First 2 = digits, 3rd = number of zeros. 103 = 10×10³ = 10 kΩ
4-digit (e.g. 1002): First 3 = digits, 4th = zeros. 1002 = 100×10² = 10 kΩ
R notation (e.g. 4R7): R replaces decimal point. 4R7 = 4.7 Ω
EIA-96 (e.g. 01C): 2-digit code + letter multiplier for 1% precision parts
