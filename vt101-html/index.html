DEC Terminal Timeline
1970
VT05 — DEC's first video terminal, 20×72 characters, no cursor addressing, no scrolling
1974
VT50 / VT52 — cursor-addressable terminals with DEC-specific escape sequences; VT52 becomes a de facto standard
1978
VT100 — the landmark model. First DEC terminal to implement ANSI X3.64 escape codes. 80/132 column mode, smooth scroll, 24 lines. Wildly successful — cloned by every major vendor. The basis for xterm and virtually every modern terminal emulator.
1982
VT101 — stripped-down VT100 compatible terminal at a lower price point. No smooth scroll, no downloadable character sets, but fully ANSI X3.64 compliant. Popular in universities and research labs.
1983
VT102 — VT101 with built-in printer port and AVO (Advanced Video Option) — double-width/height characters
1987
VT220 / VT240 — 8-bit character sets, PC keyboard layout, 132-column support standard, ReGIS graphics on VT240
1993
VT420 — DEC's last major text terminal. Dual sessions, 4-page memory, VT52/VT100/VT220 compatibility modes
Today
Living legacy — xterm, iTerm2, Windows Terminal, Alacritty, kitty — all emulate VT100/VT220 and respond to the same escape sequences defined in the early 1980s
DEC Terminal Models
VT52
1974 · ~$1,500
Cursor-addressable glass TTY. Non-ANSI, proprietary escape sequences. Still emulated by many terminals today.
Display: 24×80
Baud: Up to 9600
Screen: 12" P4 phosphor
VT100
1978 · ~$1,500
The most influential terminal ever made. First ANSI X3.64 compliant DEC terminal. 80/132 column modes, smooth scrolling, graphics characters (box drawing).
Display: 24×80 / 14×132
Baud: Up to 19,200
Screen: 12" P4 phosphor
VT101
1982 · ~$800
Budget VT100 — no smooth scroll, no AVO, no downloadable chars. ANSI compatible. Popular in labs due to lower price. Amber phosphor option available.
Display: 24×80
Baud: Up to 9600
Screen: 12" amber option
VT220
1987 · ~$900
8-bit control characters, PC-style keyboard with numeric pad and function keys F1-F20. National Replacement Character Sets for international use.
Display: 24×80 / 24×132
Baud: Up to 38,400
Interface: RS-232 + printer
VT320
1987
Enhanced VT220 with 25-line display, customizable function keys, session setup storage in NVRAM. Tilt/swivel monitor.
Display: 25×80 / 25×132
Memory: NVRAM settings
VT420
1993
Dual sessions — two simultaneous host connections. 4-page screen memory, VT52/VT100/VT220 emulation modes, 14" monitor.
Display: 24-48 × 80-132
Sessions: 2 simultaneous
Ports: RS-232 + printer + AUX

Legacy Terminals vs. Modern Emulators
Hardware Terminal Specs (VT101)
Screen: 12" CRT, P4 white or amber phosphor Columns: 80 (fixed, no 132-col mode) Lines: 24 Character: 7×10 pixel cell Interface: RS-232 serial, 75–9600 baud RAM: 2KB screen memory CPU: 8085 (2MHz) Weight: ~25 lbs (11 kg) Price: ~$795 (1982) = ~$2,500 today
Modern Emulators (VT100 heirs)
xterm — X11 original, gold standard iTerm2 — macOS, tmux integration, GPU kitty — GPU, ligatures, image protocol Alacritty — Rust, GPU, speed-focused WezTerm — Rust, Lua config, multiplexer Windows Term. — GPU, Unicode, tabs foot — Wayland native, minimal Ghostty — native per-platform, fast
