Gallery — MS-DOS & the IBM PC Era
IBM PC 5150 (1981)
The original IBM Personal Computer, model 5150 — the machine that defined the PC architecture and ran the first versions of PC DOS/MS-DOS.
IBM 5150 — Original Packaging
The IBM PC 5150 in its original box — a machine that launched an industry. Its open architecture allowed clone makers to eventually dominate the market.
MS-DOS DIR Command
The iconic MS-DOS directory listing — C:\> was the gateway to everything. Every DOS user typed DIR hundreds of times per day.
MS-DOS 6.22 Floppy Disks
MS-DOS 6.22 on 3.5" floppy disks (1994) — the last standalone MS-DOS release, distributed on the physical media that defined an era.
MS-DOS Advertisement, 1981
The original MS-DOS print advertisement from 1981 — Microsoft's OS that came with the IBM PC and launched the most successful software company in history.
Compaq MS-DOS 1.12
Compaq's early MS-DOS 1.12 — Compaq was the first IBM PC clone maker, legally reverse-engineering the BIOS to run DOS and launch the clone industry.
NEC MS-DOS 3.3C
MS-DOS 3.3C for NEC PC-98 systems — DOS ran on dozens of incompatible PC architectures, each with localized versions, before the x86 standard won out.
MS-DOS 5.0 Installation Disks
MS-DOS 5.0 (1991) installation disks — this version introduced DOSKEY, EDIT, and better memory management, and was a major upgrade for millions of users.
