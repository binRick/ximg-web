Non-Inverting Amplifier

Voltage Gain (Av)—
Vout—
Input ImpedanceVery high (GΩ)
Phase0° (in-phase)
FormulaAv = 1 + R2/R1
Unity-gain buffer (voltage follower): Set R1=∞ (remove), R2=0 (short). Av=1. High Zin, low Zout. Used for impedance matching.

Non-inverting gain: Av = 1 + R2/R1. Always ≥1. Output is in phase with input. Input impedance is the op-amp's own differential input impedance (typically 1MΩ–1TΩ).

Differential Amplifier

Differential Gain—
Vout—
FormulaVout = (R2/R1) × (V2−V1)
CMRRHigh (depends on R matching)
The differential amplifier amplifies the difference between two signals and rejects common-mode noise. For high CMRR, resistors must be matched to 0.1% or better. The Instrumentation Amplifier (INA) IC is a precision 3-op-amp version commonly used in sensor interfaces.
