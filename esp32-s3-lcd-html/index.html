Build & Flash

FQBN

esp32:esp32:esp32s3:\
  USBMode=hwcdc,\
  CDCOnBoot=cdc,\
  FlashSize=16M,\
  PSRAM=opi

Serial Port

/dev/cu.usbmodem1101

Native USB CDC — no external USB-UART bridge. Flash via esptool.py.

TFT_eSPI key defines

#define ST7789_DRIVER

#define TFT_RST     3

Python env

  • All tools run from repo .venv/ — never system Python

  • Default flash image saved in backup/
