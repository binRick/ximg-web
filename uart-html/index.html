UART — Universal Asynchronous Receiver-Transmitter

UART Frame Structure

Start bit (1) → Data bits (5–9) → Parity bit (optional) → Stop bit(s) (1 or 2)

Standard: 8N1 — 8 data bits, no parity, 1 stop bit. Line is idle-high (logic 1). Start bit is always logic 0 (low).

Key Properties

TypeAsynchronous serial, full-duplex
Wires2 (TX, RX) + GND
ClockingNo clock line — uses baud rate
TopologyPoint-to-point (2 devices)
AddressingNone
Voltage levelsTTL (0/5V), 3.3V, RS-232 (±15V)
Max speed~20 Mbps (hardware-dependent)

Wiring — Cross Connections

TX on device A connects to RX on device B, and RX on device A connects to TX on device B.
Always share a common GND.

Voltage warning: 5V UART TX connected to 3.3V device RX may damage it. Use a voltage divider (2×1kΩ) or a logic level converter.

Arduino Serial Code

Basic Serial Communication

void setup() {
  Serial.begin(115200);  // init hardware UART
  Serial.println("Hello UART");
}

void loop() {
  // Echo received bytes back
  if (Serial.available()) {
    byte b = Serial.read();
    Serial.print("Got: 0x");
    Serial.println(b, HEX);
  }
}

SoftwareSerial (any pins)

#include <SoftwareSerial.h>

// SoftwareSerial(RX_pin, TX_pin)
SoftwareSerial mySerial(4, 5);

void setup() {
  Serial.begin(9600);    // hardware (USB)
  mySerial.begin(9600); // software UART
}

void loop() {
  if (mySerial.available())
    Serial.write(mySerial.read());
  if (Serial.available())
    mySerial.write(Serial.read());
}

Parse CSV from Serial

String line;

void loop() {
  while (Serial.available()) {
    char c = Serial.read();
    if (c == '\n') {
      // process complete line
      int comma = line.indexOf(',');
      float a = line.substring(0, comma).toFloat();
      float b = line.substring(comma+1).toFloat();
      Serial.print(a+b);
      line = "";
    } else line += c;
  }
}

Parity Explained

No parity (N) — no extra bit. Most common. Use CRC or application-level error checking.
Even parity (E) — parity bit set so total 1-bits in data+parity is even.
Odd parity (O) — parity bit set so total 1-bits is odd.

Example: data 0b01000001 ('A') has two 1-bits. Even parity bit = 0. Odd parity bit = 1.
